Các thẻ HTML dành cho lập trình và văn bản kỹ thuật

1. Thẻ <kbd> – Phím bàn phím

Nhấn Ctrl + S để lưu tài liệu.

2. Thẻ <samp> – Kết quả chương trình

Ví dụ đầu ra:

Lỗi: Tệp không tồn tại!

3. Thẻ <code> – Đoạn mã lập trình

Đây là một dòng mã đơn:

console.log("Hello, world!");

4. Thẻ <var> – Biến toán học/lập trình

Diện tích hình tròn: A = π × r2

5. Thẻ <pre> – Văn bản định dạng sẵn

Đoạn mã nhiều dòng được giữ nguyên định dạng:

function sayHello(name) {
  console.log("Hello, " + name + "!");
}
sayHello("Phúc");
